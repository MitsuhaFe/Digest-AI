## 🚀 3 步开始

### 1. 更新扩展
```
chrome://extensions/
→ 移除旧版本  
→ 加载已解压的扩展程序
→ 选择: D:\CodeProject\PBL2\Digest AI\dist
→ 确认版本: 0.5.0 ✅
```

### 2. 打开 PDF
```
访问任意 PDF 文件
例如: https://example.com/document.pdf
或本地 PDF 文件
```

### 3. 提取内容
```
1. 点击扩展图标
2. 显示: "提取 PDF 文档内容中..." ✨  
3. 等待 5-15 秒
4. 查看 AI 摘要 ✅
```
