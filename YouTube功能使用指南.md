# 🎬 YouTube 功能使用指南

## 🚀 快速开始

### 1. 更新扩展

```
chrome://extensions/
→ 移除旧版本
→ 加载已解压的扩展程序
→ 选择: D:\CodeProject\PBL2\Digest AI\dist
→ 确认版本: 0.4.0 ✅
```

### 2. 使用步骤

```
1. 打开 YouTube 视频
   例如: https://www.youtube.com/watch?v=...

2. 点击扩展图标

3. 显示: "提取 YouTube 视频内容中..." ✨

4. 等待完成

5. 查看摘要 ✅
```

## ✨ 支持的功能

### 自动提取内容

- ✅ **视频字幕** - 优先中文，其次英文
- ✅ **热门评论** - 无字幕时提取
- ✅ **视频简介** - 完整描述
- ✅ **视频信息** - 标题、作者、时长

```

**无字幕：**
```

## ❓ 常见问题

**Q: 为什么有些视频提取不到字幕？**

A: 可能的原因：
- 视频没有上传字幕
- 字幕被禁用
- 网络问题

不用担心！会自动提取评论作为补充 ✅

**Q: 评论提取需要多久？**

A: 通常 1-2 秒
- 需要等待页面加载评论
- 自动提取前 10 条热门评论

**Q: 支持哪些语言的字幕？**

A: 优先级排序：
1. 中文（简体/繁体）
2. 英文
3. 其他任何可用语言
