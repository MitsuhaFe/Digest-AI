## 🚀 3 步开始

### 1. 更新扩展
```
chrome://extensions/
→ 移除旧版本
→ 加载已解压的扩展程序
→ 选择: D:\CodeProject\PBL2\Digest AI\dist
→ 确认版本: 0.4.0 ✅
```

### 2. 打开 YouTube
```
访问任意 YouTube 视频
例如: https://www.youtube.com/watch?v=...
```

### 3. 保存并查看
```
1. 点击扩展图标
2. 显示: "提取 YouTube 视频内容中..." ✨
3. 等待完成
4. 查看精彩摘要！✅
```
