# Digest AI - 项目总结

## 🎯 项目概述

**Digest AI** 是一款 Chrome 浏览器扩展程序，利用 AI 技术帮助用户快速消化和管理"稍后阅读"列表。当前已完成 MVP 版本的完整开发。

### 核心价值

- ⏱️ **节省 80% 阅读时间**：AI 自动生成摘要和核心观点
- 🎯 **精准筛选内容**：快速判断文章价值，避免低价值阅读
- 🔧 **高度可定制**：支持多种 AI 模型，满足不同需求
- 📚 **知识管理**：标签系统 + 全文搜索，构建个人知识库

## 📁 项目结构

```
digest-ai-extension/
├─ icons/                      # 图标资源（需要添加）
├─ scripts/                    # 核心脚本
│  ├─ background.js           # 后台服务
│  ├─ content.js              # 内容提取
│  ├─ readability-loader.js   # Readability 加载器
│  └─ models/                 # AI 模型适配器
│     ├─ adapter.js           # 基类
│     ├─ gemini.js            # Gemini 适配器
│     ├─ openai.js            # OpenAI 适配器
│     └─ deepseek.js          # DeepSeek 适配器
├─ styles/                     # 样式文件
│  ├─ popup.css
│  ├─ dashboard.css
│  └─ settings.css
├─ popup.html/js              # 弹出界面
├─ dashboard.html/js          # 阅读库
├─ settings.html/js           # 设置页面
├─ manifest.json              # 扩展配置
├─ webpack.config.js          # 构建配置
├─ package.json               # 依赖管理
└─ 文档/                      # 各类文档
```
