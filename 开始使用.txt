📋 快速开始 - 5 步启动指南
═══════════════════════════════════════════════════════════

第 1 步：安装依赖
────────────────
在项目目录打开终端，运行：
  npm install

第 2 步：创建临时图标
────────────────
⚠️ 这是必需的！没有图标无法加载扩展
方法 1（最快）：
  找任意一张 PNG 图片，复制三次：
  copy 你的图片.png icons\icon16.png
  copy 你的图片.png icons\icon48.png
  copy 你的图片.png icons\icon128.png

方法 2：
  查看 "创建临时图标.md" 了解更多方法

第 3 步：构建项目
────────────────
  npm run build
  
  这会在 dist\ 目录生成可加载的扩展程序

第 4 步：加载到 Chrome
────────────────
  1. 打开 Chrome，访问 chrome://extensions/
  2. 开启右上角的"开发者模式"
  3. 点击"加载已解压的扩展程序"
  4. 选择项目的 dist\ 目录
  5. 看到 Digest AI 图标！

第 5 步：配置 API Key
────────────────
  1. 点击工具栏的 Digest AI 图标
  2. 点击右上角齿轮图标进入设置
  3. 选择 AI 模型（推荐 Gemini）
  4. 粘贴您的 API Key
  5. 保存设置
