# 📄 PDF 功能使用指南

## 🚀 快速开始

### 1. 更新扩展

```
chrome://extensions/
→ 移除旧版本
→ 加载已解压的扩展程序
→ 选择: D:\CodeProject\PBL2\Digest AI\dist
→ 确认版本: 0.5.0 ✅
```

### 2. 打开 PDF 文件

**方式：**
- 📎 网络 PDF：`https://example.com/document.pdf`
- 💾 本地 PDF：`file:///C:/Users/.../document.pdf`
- 🌐 在线查看器：Chrome PDF Viewer

### 3. 提取内容

```
1. 点击扩展图标
2. 显示: "提取 PDF 文档内容中..." ✨
3. 等待 5-15 秒
4. 查看AI摘要 ✅
```

## 📊 支持说明

### ✅ 支持

- 文本型 PDF
- 最多 50 页
- 元数据完整的 PDF

### ❌ 不支持

- 扫描版 PDF（图片）
- 加密 PDF（需要密码）
- 纯图像内容
